# Portrait Experience Event Cambodia Website

This is a free one-page landing website for AAL x KINGs Cambodia.

## How to publish for free on GitHub Pages
1. Create a GitHub account.
2. Create a new repository named `portrait-experience-cambodia`.
3. Upload all files inside this folder.
4. Go to Settings > Pages.
5. Under Source, choose `Deploy from a branch`.
6. Choose branch `main` and folder `/root`.
7. Save. GitHub will give you a free website link.

## What to edit
- `index.html`: wording, booking link, Telegram link.
- `style.css`: design, layout, color.
- `assets/`: replace or add more portrait photos.
